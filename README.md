# PayBhama: Rajasthan Edition 🚩

**PayBhama** is a premium, modern digital payment application designed with a core mission: **Empowering the women of Rajasthan through financial inclusion.** 

By blending high-end fintech design with regional cultural values, PayBhama provides a secure, simple, and trusted platform for managing daily transactions in the Indian Rupee (₹).

---

## ✨ Key Features

- **🏆 Premium Rajasthan UI:** A tailor-made interface inspired by the vibrant colors of Rajasthan (Jodhpur Indigo, Jaipur Pink, Royal Saffron).
- **💸 Real-time Transactions:** Instant fund transfers using Firebase Firestore with atomic transaction logic.
- **📊 Bahi-Khata (History):** A beautiful transaction ledger showing total expenses and a categorical history grid.
- **🚀 Onboarding Walkthrough:** A clickable, 3-slide transition system explaining the app's mission to new users.
- **🔐 Secure Authentication:** Robust Login and Registration flow featuring Firebase Auth and modern material design.
- **🌓 Smooth Transitions:** Fluid screen-to-screen animations for a professional mobile experience.

---

## 🛠 Tech Stack

- **Language:** Kotlin (100%)
- **Architecture:** MVVM (Model-View-ViewModel) with Clean Architecture principles.
- **Backend:** Firebase (Authentication & Firestore).
- **UI Components:** ViewBinding, Navigation Component, Material Design 3, CoordinatorLayout.
- **Currency Support:** Indian Rupee (₹) with Indian numbering formatting.

## 🚀 Getting Started

### Prerequisites
- Android Studio Iguana or newer.
- A Firebase project with Firestore and Email/Password Auth enabled.

### 🔑 Firebase Setup (Required)
For security reasons, this repository does not include secret API keys. To run the app:
1. Go to the [Firebase Console](https://console.firebase.google.com/).
2. Create a new project and add an Android app with package name `com.paybhama.app`.
3. Download the `google-services.json` file.
4. Place the file in the **`app/`** directory of this project.

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/HARSH-1607/PayBhama.git
   ```
2. Place your `google-services.json` in the `app/` directory as described above.
3. Build and run the project on your emulator or physical device.
